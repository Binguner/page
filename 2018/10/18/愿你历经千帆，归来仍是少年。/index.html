愿你历经千帆，归来仍是少年。

Posted on 2018-10-18 | In 公众号发文 | Visitors
昨天，我们团队里的一位学弟离开了团队...

昨天，我们团队里的一位学弟离开了团队，我这人也不太会说话，好多话也不知道从何说起，只能在这里写出来了。

从今年开学之后，陆陆续续看到成员群里有人退出的提示，我心里都有些不是滋味，但看到他退出的消息之后，我心里更不是滋味了。

从去年面试他和其他新生进入团队开始，然后一轮一轮的裁人，负责面试的我们这些学长学姐内心无时无刻不在纠结，因为我知道我们的每一个决定，很可能就会影响他们整整大学四年的生活，甚至是未来的路。对于他们每一个人的去留，我们都经过了一次又一次的考虑，相信他在这次招新的过程中也有这样的体会。

团队不养懒人，技术团队更是如此，在一轮又一轮的考核工作中，我惋惜那些提前放弃，自己退出的人，为什么不能再坚持一下？也许多坚持一下就成功了呢？

在这里我想对那些曾经被我们筛选出去的同学说：「对不起，紧张的考核期限使我们很难去了解你们每一个人，我希望未来的道路你们能走的更好！」

在我们 2018 年寒假的第一次培训，虽然我在做 Android，但 Java 也略懂一点，于是我和其他几个同学共同负责 Java 组的培训工作。从 Java 基础开始，帮他们从头学起。再到后来开始学习服务端的技术，虽然我没有办法继续帮他们提升技术，但是能看到大家一点一滴的成长， 我依然很开心。

那个冬天是我过的最冷的一个冬天，我还记得有天晚上，我一个人在宿舍冷到半夜起来把舍友留下的被子加在自己身上。

但是因为他们，值了。

然后最难熬的日子就是今年暑假。我知道几台风扇根本吹不走盛夏的炎热，我也知道因为宿舍刷漆味道太浓，好多人没办法回去住，便干脆住在了办公室里，但大家都没有说过一句苦，一句累…

他们的家长看了一定会心疼，因为我也心疼。

到如今最难熬的日子都过去了，他却…

每个人离开的原因千千万，万般不舍也无奈。

在不同的阶段我们会面临不同的选择，人生就是无数选择的结果。要想从容选择，就必须学会放弃。只要是适合自己的就好。

天下没有不散的宴席。

没人会在一个团队待一辈子，我只希望他和所有成员在团队的日子里收获到了快乐和知识。

在暑假的时候我很意外地发现他和我一样都在默默的坚持锻炼。到了大学一个人还能坚持锻炼的，我认为都是很厉害的人。

不出我所料，他真的很厉害。

他不适合参加体育训练，但他硬是参加完军训，并且还是优秀学员。

他还参加了每天起早贪黑的国旗护卫队，并且成了最棒扬旗手。

离开团队对于他来说是一个新的起点，如果继续选择技术路线，希望他能加倍努力，大家现在的技术水平依然需要极大的提高；如果他选择考研或者其他方向，也希望他能加倍努力，他是我们团队出来的人，不可以给我们团队丢脸。

也希望他能保持善良，认真对待周围的同学和朋友，因为这会是我们每个人大学四年最珍贵的财富。

当初面试的时候，每个人都会被问到「技术能力、人际关系、忠诚度哪个最重要？」我现在想对他说：

你自己最重要。

愿你历经千帆，归来仍是少年。

Binguner
2018 年 10 月 18 日
