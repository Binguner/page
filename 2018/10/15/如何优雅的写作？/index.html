如何优雅的写作？

Posted on 2018-10-15 | In 公众号发文 | Visitors
你还在用 word 写文章？你还在用 word 排版？你 out 了！

写作能力需要日积月累的提升，但写作方式在你读完这篇文章之后就可以改变！

在我以前的文章「为什么我建议你们去写作」中，我已经强调过了写作的重要性。写作对个人能力的提升，对个人品牌的建立都有显著的作用。

工欲善其事必先利其器，想要提升自己的写作效率，你必须先选择一款适合写作的工具。相信身边有很多的人还在用 word 作为自己的写作工具，我想说的是 word 并不适合用来写作，word 的功能太多了，对于我们偶尔码字的人来说，其中 90% 的功能都不一定用得到，剩下 10% 的功能在使用时也有一定的学习成本。

并且在目睹身边一系列因为电脑故障，没来得及保存导致文件丢失的惨案之后，我在这里真心向大家推荐一些更简介、方便的写作工具：

  • 简书

    网页端的写作工具，在移动端也有，支持边写边浏览，并且实时自动保存。

  • 印象笔记

    老牌的笔记软件了，有什么新想法，记下来吧！

  • 有道云笔记

    和印象笔记一样。

  • Typora

    在 Windows 平台上备受好评，Mac 上也能使用，界面简洁，所见即所得，所有的效果会及时地渲染在桌面上。

  • MWeb

    Mac 上的一款良心写作软件，支持文档管理和管理博客。

  • Sublime

    升级版的记事本，有丰富的插件可以使用。

用上面的这些工具能极大的提升我们的写作效率，但这不是我写这篇文章的主要目的，我之所以推荐了上述的工具，主要是因为它们都支持一款非常方便的标记语言「Markdown」。

大家不要被标记语言这几个字所迷惑，Markdown 相较于 HTML 的便签，日常使用所需要的语法十分简单，花上几分钟学一学，可以让你一劳永逸！从此把所有的精力专注于写作，不需要额外花费心思考虑排版的问题。

如果大家不放心 Markdown 的效果，可以来看一下我的博客，里面的文章都使用了 Markdown 语法。

什么是 Markdown

John Gruber 在 2004 年创造了 Markdown 语言，这门语言的目的是希望大家使用「易于阅读、易于撰写的纯文字格式，并选择性的转换成有效的XHTML（或是HTML）」。它没有 HTML 那种复杂的标签，可以直接在字面上进行阅读。Markdown 最初在程序员的圈子中扩散，现在已经被越来越多的写作人员使用。简书、印象笔记、有道云笔记、为知笔记、CSDN、Github、Segmentfault、Wordpress 等都已经支持 Markdown。
