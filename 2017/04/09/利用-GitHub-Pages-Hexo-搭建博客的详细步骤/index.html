利用 GitHub Pages + Hexo 搭建博客的详细步骤

Posted on 2017-04-09 | In GitHub | Visitors
教你怎么搭建博客！

需要用到的工具：

  • Git
  • Node.js
  • Hexo

Git 是一个开源的分布式版本控制系统。
Node.js 是一个基于 Chrome JavaScript 运行时建立的平台， 用于方便地搭建响应速度快、易于扩展的网络应用。
Hexo 是一个简洁高效的博客框架（支持Markdown）。

现在开始搭建博客

在 Github 中创建项目

登陆 Github 账号
点击 New repository 建立一个工程仓库

在 Repository name 中填写「name.github.io」
注意将 name 改成 Owner 的名字
然后直接点击 Create repository

安装工具

  • 首先安装 Git，去官网下载对应的版本。
  • 然后安装 Node.js
    • Windows 用户
      去官网下载安装。
    • Mac 用户（没试验过）
      打开「终端」 分别执行以下两行命令。
